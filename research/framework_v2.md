## The 18-Step Protocol

This framework ensures every research output is "Exhaustive and Extensive," moving from raw data to production-ready "Frameworks."

### Phase I: Ingestion (Steps 1-3)
1.  **Define Scope:** Select the Director/Topic and the specific filmography window.
2.  **Ingest Primary Sources:** View/Read the films. (Simulated via knowledge base).
3.  **Ingest Secondary Sources:** Criticism, interviews, academic analysis.

### Phase II: Analysis (Steps 4-7)
4.  **Analyze Thematic Resonance:** Identify the core "Engine" (e.g., Body Horror, Cosmic Dread).
5.  **Map Narrative Structures:** Linear, Cyclical, Dream-Logic, Non-Linear.
6.  **Identify Visual Motifs:** Recurring imagery (The Red Room, The Double).
7.  **Cross-Reference Stylistics:** Connect to existing PHANTASMATA axes ($ARGENTO, $LYNCH, $KON).

### Phase III: Synthesis (Steps 8-13)
8.  **Synthesize Findings:** Draft the "Executive Summary."
9.  **Define Research Scope (Refined):** Narrow down to the most "Useful" elements for GRINDER.
10. **Extract Key Themes:** Create the "Concept List."
11. **Document Structural Archetypes:** Create the "Plot Blueprint."
12. **Catalog Visual Signatures:** Create the "Aesthetic Palette."
13. **Correlate Style with Theme:** Explain *why* the style works (Epistemology).

### Phase IV: Framework Generation (Steps 14-18)
14. **Develop Framework Schema:** Create a named "System" for this director (e.g., The "New Flesh" Protocol).
15. **Outline Application:** How to use this in PHANTASMATA (e.g., "Use for Body Horror sequences").
16. **Generate Example:** Write a short fragment using this framework.
17. **Refine Framework:** Adjust based on the example's success/failure.
18. **Finalize Documentation:** Publish the `.md` report.

## Research Clusters (The Roadmap)

To manage the massive scope, directors are grouped into thematic clusters.

### Cluster 1: The New Flesh (Body & Transgression)
*   **Focus:** Biological horror, physical transformation, abjection.
*   **Targets:** David Cronenberg, Lars Von Trier, Eli Roth, Julia Ducournau.
*   **PHANTASMATA Alignment:** $GRINDHOUSE / Week 4 "Throats."

### Cluster 2: The Cold Gaze (Structure & Formalism)
*   **Focus:** Symmetry, detachment, systems, control.
*   **Targets:** Stanley Kubrick, Yorgos Lanthimos, Michael Haneke, David Fincher.
*   **PHANTASMATA Alignment:** $LYNCH / The "Mundane Anchor."

### Cluster 3: The Ghost within the Machine (Spirit & Psychology)
*   **Focus:** Madness, repression, the supernatural as psychological.
*   **Targets:** Ingmar Bergman, Roman Polanski, Kaneto Shindo, Andrzej Żuławski.
*   **PHANTASMATA Alignment:** $PSYCHO_SUBCONSCIOUS / $KON.

### Cluster 4: The Mechanism (Slasher & Metaphor)
*   **Focus:** The killer, the rules, the final girl, the suburbs.
*   **Targets:** John Carpenter, Wes Craven, Tobe Hooper, Alfred Hitchcock.
*   **PHANTASMATA Alignment:** $GIALLO / $FOUND_FOOTAGE.

### Cluster 5: The Folk & The Cosmic (Ritual & Dread)
*   **Focus:** Paganism, daylight horror, deep time.
*   **Targets:** Ari Aster, Robert Eggers, Masaki Kobayashi, Ben Wheatley.
*   **PHANTASMATA Alignment:** The "Deep Drift."
